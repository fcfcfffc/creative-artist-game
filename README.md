# 创作中的艺术家 - 思维创意收集游戏

一个有趣的创意收集游戏，通过点击画布激发创意，开启宝箱获得不同等级的创意作品，并在思维宫殿中查看收藏。

## 🎮 游戏特色

- **创意工坊**: 点击画布进行创作，100次点击获得1把创意钥匙
- **创意宝箱**: 使用钥匙开启宝箱，获得不同稀有度的创意作品
- **思维宫殿**: 查看收藏的创意作品，点击卡片获得AI生成的创作方案
- **每日任务**: 完成打扫任务获得额外钥匙奖励
- **支付系统**: 支持支付宝/微信支付充值创意钥匙

## 🎨 稀有度系统

- **普通** (40%): 基础创意作品
- **稀有** (30%): 有价值的创意灵感
- **史诗** (20%): 杰出的创意构思
- **传说** (8%): 卓越的艺术理念
- **神话** (2%): 传奇级创意杰作

## 🤖 AI创作方案

根据不同稀有度的作品，AI会生成相应级别的创作建议：
- 从基础技法到革命性创新
- 包含详细的创作指导和相关标签
- 帮助用户获得创作灵感

## 📱 设备支持

## 📦 本地运行

1. 克隆仓库
```bash
git clone https://github.com/your-username/creative-artist-game.git
```

2. 进入项目目录
```bash
cd creative-artist-game
```

3. 使用本地服务器运行（推荐）
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 或使用 Live Server (VS Code 扩展)
```

4. 在浏览器中访问 `http://localhost:8000`

## 🎯 游戏玩法

1. **注册账号**: 输入2-10个字符的昵称开始游戏
2. **创作获取钥匙**: 在创意工坊点击画布100次获得1把钥匙
3. **开启宝箱**: 使用钥匙开启创意宝箱获得作品
4. **查看收藏**: 在思维宫殿查看和管理收藏的作品
5. **AI创作方案**: 点击作品卡片获得AI生成的创作建议
6. **每日任务**: 完成打扫任务获得额外奖励
